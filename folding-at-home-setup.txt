19:T1bf4,<hr>
<p>The world is facing an unprecedented situation with the <a href="https://www.who.int/emergencies/diseases/novel-coronavirus-2019">Coronavirus outbreak</a>. It is crucial at these times for everyone to join hands and fight the rapidly spreading virus. Computing power available to researchers is very valuable right now. <a href="https://foldingathome.org/">Folding@Home</a> is a project which allows crowd-sourcing computing power by the help of distributed computing over public compute nodes for the study of biological proteins. The project is currently providing (at the time of writing this article) more than 470 petaFlops of computing power to researchers for studying Coronavirus, and to eventually prepare a cure for it.</p>
<h2>But, What should I do?</h2>
<blockquote>
<p>Contribute.</p>
</blockquote>
<p>Your computer&#39;s idle time can be utilized towards contributing to the Folding@Home project very, very easily.
<img src="https://digitalpress.fra1.cdn.digitaloceanspaces.com/4ml9m8u/2025/04/image-5.png" alt=""></p>
<h2>What do I need?</h2>
<p>All that you need for contributing to the Folding@Home project -</p>
<ul>
<li>A working computer, can be local, can be a VM</li>
<li>A stable internet connection</li>
<li>A will to help, even if for a few hours a day!</li>
</ul>
<p>Do you have these? If yes (or even if no), read on!</p>
<h2>How to setup?</h2>
<p>I shall provide a very quick but complete setup here that should get you started in no time.</p>
<p>The general procedure is as follows -</p>
<ul>
<li><a href="https://apps.foldingathome.org/passkey/create">Create a passkey</a> for yourself.</li>
<li>Download and install the <code>FAHClient</code> setup on the machine which will perform computation. You&#39;ll need your passkey in this step.</li>
<li>Download the install the <code>FAHControl</code> setup on the machine which you will use to control and monitor your <code>FAHClient</code>.</li>
<li>Any additional configuration as needed.</li>
</ul>
<p>Let&#39;s get started!</p>
<h3>Setup on a local machine</h3>
<p>Setting up on a local machine is very simple. Follow along the steps below -</p>
<h4>Windows</h4>
<ul>
<li><a href="https://download.foldingathome.org/releases/public/release/fah-installer/windows-10-32bit/v7.5/fah-installer_7.5.1_x86.exe">Download</a> and run the installer.</li>
<li>Click Yes, Next, I Agree, Next, Finish.</li>
<li>Enter a name, <code>team number 250120</code> (to join our team, use 0 for no team), and Passkey.</li>
<li>Click Save.</li>
</ul>
<h4>Linux</h4>
<ul>
<li>Download the <code>FAHClient</code> package for your distro, using the following command:</li>
</ul>
<pre><code class="language-bash"># Debian/Ubuntu/Mint
wget https://download.foldingathome.org/releases/public/release/fahclient/debian-testing-64bit/v7.4/fahclient_7.4.4_amd64.deb

# RedHat / CentOS / Fedora
wget https://download.foldingathome.org/releases/public/release/fahclient/centos-5.3-64bit/v7.4/fahclient-7.4.4-1.x86_64.rpm
</code></pre>
<ul>
<li>Install package using following command:</li>
</ul>
<pre><code class="language-bash"># Debian/Ubuntu/Mint
sudo dpkg -i --force-depends fahclient_7.4.4_amd64.deb
sudo apt install -f

# RedHat / CentOS / Fedora
sudo rpm -i --nodeps fahclient-7.4.4-1.x86_64.rpm
</code></pre>
<ul>
<li>Enter a name, <code>team number 250120</code> (to join our team, use 0 for no team), and Passkey.</li>
<li>Complete the <code>FAHClient</code> installation.</li>
<li>Download <code>FAHControl</code> using the following command:</li>
</ul>
<pre><code class="language-bash"># Debian/Ubuntu/Mint
wget https://download.foldingathome.org/releases/public/release/fahcontrol/debian-testing-64bit/v7.4/fahcontrol_7.4.4-1_all.deb

# RedHat / CentOS / Fedora
wget https://download.foldingathome.org/releases/public/release/fahcontrol/centos-5.3-64bit/v7.4/fahcontrol-7.4.4-1.noarch.rpm
</code></pre>
<ul>
<li>Install FAHControl using the following command:</li>
</ul>
<pre><code class="language-bash"># Debian/Ubuntu/Mint
sudo dpkg -i --force-depends fahcontrol_7.4.4-1_all.deb
sudo apt install -f

# RedHat / CentOS / Fedora
sudo rpm -i --nodeps fahcontrol-7.4.4-1.noarch.rpm
</code></pre>
<h3>Setup on a VM</h3>
<p>After provisioning your VM, assuming it is a Linux VM (we prefer Ubuntu 18.04 LTS), follow the given steps -</p>
<h4>Steps to be performed on VM</h4>
<ul>
<li>Update your package repository.</li>
</ul>
<pre><code class="language-bash">sudo apt update
</code></pre>
<ul>
<li>Download the <code>FAHClient</code> package for your distro, using the following command:</li>
</ul>
<pre><code class="language-bash"># Debian/Ubuntu/Mint
wget https://download.foldingathome.org/releases/public/release/fahclient/debian-testing-64bit/v7.4/fahclient_7.4.4_amd64.deb
</code></pre>
<ul>
<li>Install package using the following command:</li>
</ul>
<pre><code class="language-plaintext"># Debian/Ubuntu/Mint
sudo dpkg -i --force-depends fahclient_7.4.4_amd64.deb
sudo apt install -f
</code></pre>
<ul>
<li>Enter a name, <code>team number 250120</code> (to join our team, use 0 for no team), and Passkey.</li>
<li>Complete the <code>FAHClient</code> installation.</li>
<li>Stop the <code>FAHClient</code></li>
</ul>
<pre><code class="language-bash">sudo /etc/init.d/FAHClient stop
</code></pre>
<ul>
<li>Edit the configuration file, as shown below:</li>
</ul>
<pre><code class="language-bash">sudo nano /etc/fahclient/config.xml
</code></pre>
<p>Your file should look like:</p>
<pre><code class="language-text">&lt;config&gt;
  &lt;!-- Client Control --&gt;
  &lt;fold-anon v=&#39;true&#39;/&gt;

  &lt;!-- Folding Slot Configuration --&gt;
  &lt;gpu v=&#39;false&#39;/&gt;

  &lt;!-- HTTP Server --&gt;
  &lt;allow v=&#39;127.0.0.1 your_local_system_ip&#39;/&gt;

  &lt;!-- Slot Control --&gt;
  &lt;power v=&#39;full&#39;/&gt;

  &lt;!-- User Information --&gt;
  &lt;passkey v=&#39;your_passkey&#39;/&gt;
  &lt;team v=&#39;250120&#39;/&gt;
  &lt;user v=&#39;your_name&#39;/&gt;

  &lt;!-- Web Server --&gt;
  &lt;web-allow v=&#39;127.0.0.1 your_local_system_ip&#39;/&gt;

  &lt;!-- Folding Slots --&gt;
  &lt;slot id=&#39;0&#39; type=&#39;CPU&#39;/&gt;
&lt;/config&gt;
</code></pre>
<ul>
<li>Restart the <code>FAHClient</code></li>
</ul>
<pre><code class="language-bash">sudo /etc/init.d/FAHClient start
</code></pre>
<h4>Steps to be performed locally</h4>
<ul>
<li>Now, you&#39;ll need to install the <code>FAHControl</code> on your local system. Use the steps given in the Setup on local machine section to install <code>FAHControl</code> on your local system.</li>
<li>You should be able to access your remote <code>FAHClient</code> using the following address:</li>
</ul>
<pre><code class="language-bash">http://your_vm_ip:7396
</code></pre>
<p>You should see the dashboard displaying information as shown in the following screenshot:
<img src="https://digitalpress.fra1.cdn.digitaloceanspaces.com/4ml9m8u/2025/04/image-6.png" alt="">
You can always head over the <a href="https://foldingathome.org/support/faq/">official installation documentation</a> for more details and options! You can also ask questions on the <a href="https://foldingforum.org/">Folding Forum</a>.</p>
