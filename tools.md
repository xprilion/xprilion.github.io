# Tools

Over the years, I have built a bunch of tools which help me on an everyday basis. Here are some which you can have a go at:
- [**LinkBorg**](https://linkb.org/)
*Purpose:* Link shortener and single page profile with free, open API
*Pricing:* Free!
- [**Mintro.CSS**](https://xprilion.com/mintrocss/)
*Purpose:* Minimal Retro CSS stylings for your web pages, classless, under 5kb
*Pricing:* Free!
- [**QrBorg**](https://qrborg.xprilion.workers.dev/)
*Purpose:* Generate QR codes easily from a single API call or from within the browser
*Pricing:* Free!
- [**Random Project Name Generator API**](https://project-name-generator-api.xprilion.com/)
*Purpose:* Generate a random project name similar to the ones on Github/Repl.it via a single API call
*Pricing:* Free!
- [**Vectorize.sh**](https://vectorize.sh/)
*Purpose:* Documents to vector store within seconds, with a ready-to-hit Swagger API documentation
*Pricing:* Free (up to a document size limit)
- [**LlamaStudio**](https://llamastudio.dev/)
*Purpose:* Convert prompts to APIs within seconds
*Pricing:* Free within limits
