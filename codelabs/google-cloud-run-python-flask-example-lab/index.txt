14:Tc1f,<h2>Overview</h2>
<p>Duration: 1</p>
<p>Hey there!</p>
<p>This codelab will walk you through the process of creating a simple "Hello World" application using Python Flask and deploying it to Google Cloud Run.</p>
<h2>Requirements</h2>
<p>Duration: 2</p>
<p>In order to follow this codelab, you'll need the following:</p>
<ul>
<li>A Google Cloud Platform account.</li>
<li>Google Cloud CLI installed and initialized.</li>
<li>Python installed locally.</li>
</ul>
<h2>Setting Up Your Google Cloud Account</h2>
<p>Duration: 10</p>
<ul>
<li>Head over to <a href="https://console.cloud.google.com/" rel="noopener noreferrer">Google Cloud Platform</a> and create an account.</li>
<li>Enable billing</li>
<li>Create a new project, let's name it <code>project-x</code>.</li>
</ul>
<h2>Install Google Cloud CLI</h2>
<p>Duration: 5</p>
<p>Follow the official guide to install the Google Cloud CLI: <a href="https://cloud.google.com/sdk/docs/install" rel="noopener noreferrer">Installing Google Cloud SDK</a>.</p>
<h2>Create and Run the Flask Application</h2>
<p>Duration: 15</p>
<p>Create a file named <a href="http://main.py" rel="noopener noreferrer"><code>main.py</code></a> with the following content:</p>
<pre><code class="language-python">from flask import Flask
import os

app = Flask(__name__)

@app.route('/')
def hello_world():
    return 'Hello, World!'

if __name__ == '__main__':
    app.run(debug=True, host="0.0.0.0", port=int(os.environ.get("PORT", 8080)))
</code></pre>
<p>Install Flask:</p>
<pre><code class="language-bash">pip install Flask
</code></pre>
<p>Create a Python virtual environment and activate it:</p>
<pre><code class="language-bash">python3 -m venv .venv
source .venv/bin/activate
</code></pre>
<p>Set up a new directory for your project:</p>
<pre><code class="language-bash">mkdir hello-world-cloud-run
cd hello-world-cloud-run
</code></pre>
<h2>Running the App Locally</h2>
<p>Duration: 2</p>
<ul>
<li>Open your web browser and go to <a href="http://localhost:8080" rel="noopener noreferrer"><code>http://localhost:8080</code></a>. You should see the "Hello, World!" message.</li>
</ul>
<p>Run your application locally:</p>
<pre><code class="language-bash">python main.py
</code></pre>
<h2>Deploy to Google Cloud Run</h2>
<p>Duration: 5</p>
<ul>
<li>Wait for the deployment to complete. You will receive a URL where your application is live.</li>
</ul>
<p>Deploy your application to Google Cloud Run:</p>
<pre><code class="language-bash">gcloud run deploy
</code></pre>
<p>Generate the requirements.txt file</p>
<pre><code class="language-bash">pip list --format=freeze &gt; requirements.txt
</code></pre>
<h2>Conclusion</h2>
<p>Duration: 1</p>
<p>Congratulations! You've just built and deployed a simple application on Google Cloud Run using Python and Flask!</p>
<h2>What's Next?</h2>
<p>Duration: 1</p>
<ul>
<li>Experiment with adding more routes and functionalities to your Flask application.</li>
<li>Learn more about managing and monitoring your applications on Google Cloud Run.</li>
<li>Explore other Google Cloud services that can enhance your application.</li>
</ul>
