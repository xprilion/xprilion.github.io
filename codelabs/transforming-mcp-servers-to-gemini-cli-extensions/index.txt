14:T1a66,<h2>Overview</h2>
<p>Duration: 1</p>
<p>Hey there! This codelab walks you through the steps required to convert MCP servers into extensions for Gemini CLI. We begin with creating a super simple MCP server and then convert it to an extension.</p>
<h2>Requirements</h2>
<p>Duration: 2</p>
<p>In order to follow this codelab, you'll need the following:</p>
<ul>
<li>Locally installed <a href="https://geminicli.com" rel="noopener noreferrer">Gemini CLI</a></li>
<li>Locally installed Python</li>
<li>Locally installed nodejs</li>
<li>(Optional) <a href="https://docs.astral.sh/uv/" rel="noopener noreferrer">Astral UV</a> installed locally, to run pip installs faster. If not, just remove <code>uv</code> from the <code>uv pip</code> commands.</li>
<li>(Optional) A Google Cloud Project with billing enabled.</li>
</ul>
<p>This codelab assumes you're on a Linux environment (Mac/Windows WSL/linux distros, etc.). If you're on Windows or any other operating system, you may have to substitute some of the commands to make them work.</p>
<p>For example:</p>
<pre><code class="language-bash">On Linux: 
source .venv/bin/activate 

On Windows:
1. If you're using command prompt: venv\Scripts\activate.bat
2. If you're using Powershell: .\venv\Scripts\activate.ps1
</code></pre>
<p>File paths might also differ accordingly, please check documentation of tools for best guidance specific to your system. </p>
<h2>Setup Repo</h2>
<p>Duration: 5</p>
<p>Let's setup our repository for the day.</p>
<p>We'll be working with this repository - <a href="https://github.com/xprilion/fastmcp-sample" rel="noopener noreferrer">https://github.com/xprilion/fastmcp-sample</a></p>
<p>Clone it using:</p>
<pre><code class="language-bash">git clone git@github.com:xprilion/fastmcp-sample.git
</code></pre>
<p>Now, let's set it up.</p>
<pre><code class="language-bash">uv venv
source ~/.venv/bin/activate
uv pip install -r requirements.txt
</code></pre>
<p>With this done, we're ready to setup Gemini CLI!</p>
<h2>Setup Gemini CLI</h2>
<p>Duration: 3</p>
<p>Next, we'll setup Gemini CLI.</p>
<p>Follow the installation instructions on this page - <a href="https://geminicli.com/docs/get-started/installation/" rel="noopener noreferrer">https://geminicli.com/docs/get-started/installation/</a></p>
<p>Whn done, you should be able to run the following command - </p>
<pre><code class="language-bash">gemini
</code></pre>
<p>This should bring up the following on your terminal:
<img src="https://digitalpress.fra1.cdn.digitaloceanspaces.com/4ml9m8u/2025/11/Screenshot-2025-11-29-at-2.20.29---PM.png" alt="" /></p>
<h2>Run MCP Server</h2>
<p>Duration: 2</p>
<p>For this, you'll need 2 terminal windows.</p>
<p>On the first terminal window, run the following to start the MCP server.</p>
<pre><code class="language-bash">python setup_db.py # &lt;- Run this only once, at first run, then never again

python main.py
</code></pre>
<p>This will start the server as shown below: 
<img src="https://digitalpress.fra1.cdn.digitaloceanspaces.com/4ml9m8u/2025/11/Screenshot-2025-11-29-at-2.23.47---PM.png" alt="" /></p>
<h2>MCP with Gemini CLI</h2>
<p>Duration: 2</p>
<p>Now, you can add the MCP server to your Gemini CLI instance. On a new terminal window, create a Gemini settings configuration with the following commands:</p>
<pre><code class="language-bash">mkdir .gemini
cd .gemini
nano settings.json

## Windows users may not have nano, use alternative code editor
</code></pre>
<p>And then add the following to that file:</p>
<pre><code class="language-json">{
  "mcpServers": {
    "todo-server": {
      "httpUrl": "http://localhost:8000/mcp",
      "trust": true
    }
  }
}
</code></pre>
<p>After this, if you run the following command:</p>
<pre><code class="language-bash">gemini
/mcp list # &lt;- inside the gemini shell
</code></pre>
<p>Then you should see the following tools being displayed as an active MCP server:
<img src="https://digitalpress.fra1.cdn.digitaloceanspaces.com/4ml9m8u/2025/11/Screenshot-2025-11-29-at-2.30.04---PM.png" alt="" />
In case it shows up as disconnected, retrace your steps.</p>
<h2>Deploy the MCP Server</h2>
<p>Duration: 2</p>
<p><strong>This step is optional.</strong></p>
<p>Now, you need to deploy the MCP server before you can remotely access it as an extension.</p>
<p>Run the following with <code>gcloud</code> cli installed on your system.</p>
<pre><code class="language-bash">gcloud run deploy
</code></pre>
<p>Then follow the prompts on the screen to deploy the repo.</p>
<h2>Install as extension</h2>
<p>Duration: 5</p>
<p>Head to <a href="https://github.com/xprilion/fastmcp-gemini-ext/" rel="noopener noreferrer">https://github.com/xprilion/fastmcp-gemini-ext</a></p>
<p>If you performed the deployment in the previous step, you'll need to follow the instructions on the README of the above Github repo. Else, you can simply use the existing repo. </p>
<p>Run </p>
<pre><code class="language-bash">gemini extensions install https://github.com/xprilion/fastmcp-gemini-ext
</code></pre>
<p>This will install the extension to Gemini CLI.</p>
<p>Now, you can again verify the extension installation with</p>
<pre><code class="language-bash">gemini
/extensions list
</code></pre>
<p>And get the following output</p>
<pre><code class="language-bash">&gt; /extensions list

Installed extensions:
  todo-ext (v0.1.0) - active (update available)
</code></pre>
<h2>Conclusion</h2>
<p>Duration: 5</p>
<p>Congratulations on making it here! 🎉</p>
<p>In the duration of the codelab you've learnt - </p>
<ul>
<li>How to create MCP servers with FastMCP</li>
<li>How to deploy MCP servers with Google Cloud Run</li>
<li>How to install Gemini CLI </li>
<li>How to create Gemini CLI Extensions</li>
<li>How to install and use Gemini CLI Extensions</li>
</ul>
