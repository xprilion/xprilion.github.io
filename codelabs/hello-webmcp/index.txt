14:Te60,<h2>Overview</h2>
<p>Duration: 1</p>
<p>Hey there! This codelab introduces you to WebMCP - the very early W3C  draft submitted by Google and Microsoft to enable better AI agents in the browser. In the shortest, and as per the draft report: <strong>the WebMCP API enables web applications to provide JavaScript-based tools to AI agents.</strong></p>
<p>You can read the draft report here: <a href="https://webmachinelearning.github.io/webmcp/" rel="noopener noreferrer">https://webmachinelearning.github.io/webmcp/</a></p>
<h2>Requirements</h2>
<p>Duration: 3</p>
<p>In order to follow this codelab, you'll need the following:</p>
<ul>
<li>Python on local machine</li>
<li>Chrome: Version 146.0.7672.0 or higher.</li>
<li>Gemini API Key, obtain a free one here - <a href="https://ai.dev/apikey" rel="noopener noreferrer">https://ai.dev/apikey</a></li>
</ul>
<p>Enable WebMCP on your browser:</p>
<ul>
<li>Open Chrome and navigate to chrome://flags/#enable-webmcp-testing</li>
<li>Set the flag to Enabled.</li>
<li>Relaunch Chrome to apply the changes.</li>
</ul>
<p>Additionally, you can install this extension by Google to test out WebMCP:
<a href="https://chromewebstore.google.com/detail/webmcp-model-context-tool/gbpdfapgefenggkahomfgkhfehlcenpd" rel="noopener noreferrer">WebMCP - Model Context Tool Inspector - Chrome Web StoreInspect, monitor, and execute WebMCP tools manually or with Gemini<img src="https://digitalpress.fra1.cdn.digitaloceanspaces.com/4ml9m8u/icon/icon_144px.png" alt="" />Chrome Web Store<img src="https://digitalpress.fra1.cdn.digitaloceanspaces.com/4ml9m8u/thumbnail/dwvaKnrF0HWleaF2JPEEEOBF5RJYsRislvt7HePRXJnlPL65lU2QYdUAsOej31bqgrZSFyDcRUnKlSLv_36IZzgEEQ-s128-rj-sc0x00ffffff" alt="" /></a>
This codelab assumes you're on a Linux environment (Mac/Windows WSL/linux distros, etc.). If you're on Windows or any other operating system, you may have to substitute some of the commands to make them work.</p>
<h2>Setup Repo</h2>
<p>Duration: 3</p>
<p>Let's setup our repository for the day.</p>
<p>We'll be working with this repository - <a href="https://github.com/xprilion/webmcp-train-your-baby-dragon" rel="noopener noreferrer">https://github.com/xprilion/webmcp-train-your-baby-dragon</a></p>
<p>Clone it using:</p>
<pre><code class="language-bash">git@github.com:xprilion/webmcp-train-your-baby-dragon.git
</code></pre>
<p>Switch to <code>main</code> branch</p>
<pre><code class="language-bash">git checkout main
</code></pre>
<p>Assuming you've Python on your device,</p>
<pre><code class="language-bash">python -m http.server
</code></pre>
<p>With this done, you should be able to navigate to <code>http://localhost:8000</code> to see the web page rendered.</p>
