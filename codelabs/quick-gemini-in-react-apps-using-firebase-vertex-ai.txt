15:T3576,<h2>Overview</h2>
<p>Duration: 1</p>
<p>Hey there!</p>
<p>This codelab will walk you through the process of integrating Google&#39;s Gemini into your React apps with a simple 4 step process.</p>
<h2>Requirements</h2>
<p>Duration: 2</p>
<p>In order to follow this codelab, you&#39;ll need the following:</p>
<ul>
<li>A <a href="https://cloud.google.com/">Google Cloud Platform</a> account.</li>
<li>A Google Account with <a href="https://console.firebase.google.com/">Firebase Console </a>access.</li>
<li>Access to <a href="https://firebase.studio">Firebase Studio</a></li>
<li>(Optional) Nodejs installed locally, if not using Firebase Studio.</li>
</ul>
<h2>Setting Up Your Google Cloud Account</h2>
<p>Duration: 10</p>
<ul>
<li>Head over to <a href="https://console.cloud.google.com/">Google Cloud Platform</a> and create an account.</li>
<li>Enable billing</li>
<li>Create a new project, let&#39;s name it <code>project-x</code>.</li>
</ul>
<p>If you&#39;re doing this at an in-person workshop and have access to trial credits from GCP, follow the instructions on screen to activate the account and redeem your credits.</p>
<h2>Access Firebase Console</h2>
<p>Duration: 2</p>
<p>Head to <a href="https://console.firebase.google.com/">Firebase Console </a>and accept conditions as you like them.</p>

<p>Duration: 2</p>
<p>Head to <a href="https://studio.firebase.google.com">Firebase Studio</a> and accept any T&amp;C as suitable.

<p>Duration: 5</p>

</code></pre>
<p>This will trigger a test of your faith in Firebase Studio. Choose to trust it as shown below:

<p>Next, import the Vertex AI into your code in <code>config.ts</code> . Add the following line: </p>
<pre><code class="language-typescript">// Import the functions you need from the SDKs you need
import { initializeApp } from &quot;firebase/app&quot;;
import { getVertexAI, getGenerativeModel } from &quot;firebase/vertexai&quot;; // Add this line

...
</code></pre>
<p>Now we can initialize Gemini through Vertex AI and export it. Add the following lines in the <code>config.ts</code> after initializing the <code>app</code> but before exporting it: </p>
<pre><code class="language-typescript">...

const app = initializeApp(firebaseConfig);

// Add the lines below

// Initialize FirebaseApp
const app = initializeApp(firebaseConfig);

// Initialize the Vertex AI service
const vertexAI = getVertexAI(app);

// Initialize the generative model with a model that supports your use case
const model = getGenerativeModel(vertexAI, { model: &quot;gemini-2.0-flash&quot; });

// Wrap in an async function so you can use await
export async function geminiApi(prompt: string) {
  // To generate text output, call generateContent with the text input
  const result = await model.generateContent(prompt);

  const response = result.response;
  const text = response.text();
  return text;
}

// Add the lines above

export default app;
</code></pre>
