14:T358e,<h2>Overview</h2>
<p>Duration: 1</p>
<p>Hey there!</p>
<p>This codelab will walk you through the process of integrating Google's Gemini into your React apps with a simple 4 step process.</p>
<h2>Requirements</h2>
<p>Duration: 2</p>
<p>In order to follow this codelab, you'll need the following:</p>
<ul>
<li>A <a href="https://cloud.google.com/" rel="noopener noreferrer">Google Cloud Platform</a> account.</li>
<li>A Google Account with <a href="https://console.firebase.google.com/" rel="noopener noreferrer">Firebase Console </a>access.</li>
<li>Access to <a href="https://firebase.studio" rel="noopener noreferrer">Firebase Studio</a></li>
<li>(Optional) Nodejs installed locally, if not using Firebase Studio.</li>
</ul>
<h2>Setting Up Your Google Cloud Account</h2>
<p>Duration: 10</p>
<ul>
<li>Head over to <a href="https://console.cloud.google.com/" rel="noopener noreferrer">Google Cloud Platform</a> and create an account.</li>
<li>Enable billing</li>
<li>Create a new project, let's name it <code>project-x</code>.</li>
</ul>
<p>If you're doing this at an in-person workshop and have access to trial credits from GCP, follow the instructions on screen to activate the account and redeem your credits.</p>
<h2>Access Firebase Console</h2>
<p>Duration: 2</p>
<p>Head to <a href="https://console.firebase.google.com/" rel="noopener noreferrer">Firebase Console </a>and accept conditions as you like them.</p>
<h2>Access Firebase Studio</h2>
<p>Duration: 2</p>
<p>Head to <a href="https://studio.firebase.google.com" rel="noopener noreferrer">Firebase Studio</a> and accept any T&amp;C as suitable.
<img src="https://digitalpress.fra1.cdn.digitaloceanspaces.com/4ml9m8u/2025/05/image-30.png" alt="" /></p>
<h2>Create a React Project</h2>
<p>Duration: 5</p>
<p>In Firebase Studio, click on "+ New Workspace" button to bring up the template selector page/modal. Choose "React".
<img src="https://digitalpress.fra1.cdn.digitaloceanspaces.com/4ml9m8u/2025/05/Screenshot-2025-05-04-at-10.16.05-AM.png" alt="" />
Provide any name for your workspace and select the "Typescript" radio. You can choose to go with Javascript, you dinosaur.
<img src="https://digitalpress.fra1.cdn.digitaloceanspaces.com/4ml9m8u/2025/05/Screenshot-2025-05-04-at-10.17.37-AM.png" alt="" />
Wait for the workspace creation to complete because good things take time.</p>
<p>Once done, you should see the Firebase Studio code editor interface as shown below.
<img src="https://digitalpress.fra1.cdn.digitaloceanspaces.com/4ml9m8u/2025/05/Screenshot-2025-05-04-at-10.20.24-AM.png" alt="" />
Now, we're good to start integrating Firebase Vertex AI to our application!</p>
<h2>Create a Firebase Project</h2>
<p>Duration: 5</p>
<p>Now, head to the Firebase Console at <a href="https://console.firebase.google.com/" rel="noopener noreferrer">https://console.firebase.google.com</a>.</p>
<p>Try clicking this fancy card - 
<img src="https://digitalpress.fra1.cdn.digitaloceanspaces.com/4ml9m8u/2025/05/Screenshot-2025-05-04-at-10.25.02-AM.png" alt="" />
Give your project any name as suitable.</p>
<p>Complete the next steps as you feel like, until you end up with another loading screen followed by -
<img src="https://digitalpress.fra1.cdn.digitaloceanspaces.com/4ml9m8u/2025/05/Screenshot-2025-05-04-at-10.27.13-AM.png" alt="" /></p>

<p>Duration: 3</p>
<p>Now, we need to obtain the Firebase config keys for our project.</p>
<p>Head to Project Overview &gt; Project Settings in the top left menu as shown below:
<img src="https://digitalpress.fra1.cdn.digitaloceanspaces.com/4ml9m8u/2025/05/Screenshot-2025-05-04-at-10.27.51-AM.png" alt="" />
Scroll down and Click the <code>&lt;/&gt;</code> icon to setup a web app configuration for this project.
<img src="https://digitalpress.fra1.cdn.digitaloceanspaces.com/4ml9m8u/2025/05/Screenshot-2025-05-04-at-10.28.12-AM.png" alt="" />
You should see a form like the one shown below:

<h2>Add Firebase to your project code</h2>
<p>Duration: 5</p>
<p>Head back to the editor interface in the Firebase Studio.</p>
<p>Open a new terminal at the bottom of the screen by clicking the <code>+</code> button as shown below - 
<img src="https://digitalpress.fra1.cdn.digitaloceanspaces.com/4ml9m8u/2025/05/Screenshot-2025-05-04-at-10.35.15-AM.png" alt="" />
Copy the command from the Firebase Console and run it into the new terminal: </p>
<pre><code class="language-curl">npm install firebase
</code></pre>
<p>If all goes well, you'll see a clean installation on your terminal, if not, you can see it below:

Next, create a file <code>config.ts</code> inside your <code>src</code> folder in the studio code editor.</p>
<p>Copy and paste the configuration provided in the Firebase Console. </p>
<p>At the end of the <code>config.ts</code> file, add the following line to complete it for usage.</p>

<p>On the left menu bar, click on AI &gt; Vertex AI.

<p>You'll be asked to create/select a billing account for this Firebase project.

<h2>Initalize and Integrate Firebase Vertex AI</h2>
<p>Duration: 5</p>
<p>Now we're ready to start integrating Firebase Vertex AI into our React app. </p>
<p>Head back to the Firebase Studio code editor and in the terminal, enter the following command:</p>
<pre><code class="language-bash">firebase init
</code></pre>
<p>This will trigger a test of your faith in Firebase Studio. Choose to trust it as shown below: 
<img src="https://digitalpress.fra1.cdn.digitaloceanspaces.com/4ml9m8u/2025/05/Screenshot-2025-05-04-at-10.54.03-AM.png" alt="" />
Once granted, you'll be presented with a list of your Google Cloud Projects to choose from to link with this project on Firebase Studio.</p>
<p>Choose the one we recently created!
<img src="https://digitalpress.fra1.cdn.digitaloceanspaces.com/4ml9m8u/2025/05/Screenshot-2025-05-04-at-10.54.18-AM.png" alt="" />
When done, your terminal will update to ask you which Firebase services you want to initialize for this project. 
<img src="https://digitalpress.fra1.cdn.digitaloceanspaces.com/4ml9m8u/2025/05/Screenshot-2025-05-04-at-10.56.20-AM.png" alt="" />
We don't really need any of these because there's no separate configuration required but just for the sake of completing the command, choose "Storage" and hit enter for the next steps. </p>
<p>You'll be asked to select the relevant project you want to select for your code - choose the one you just created.
<img src="https://digitalpress.fra1.cdn.digitaloceanspaces.com/4ml9m8u/2025/05/Screenshot-2025-05-04-at-10.57.47-AM.png" alt="" />
Keep hitting "Enter" until this process completes.</p>
<p>Next, import the Vertex AI into your code in <code>config.ts</code> . Add the following line: </p>
<pre><code class="language-typescript">// Import the functions you need from the SDKs you need
import { initializeApp } from "firebase/app";
import { getVertexAI, getGenerativeModel } from "firebase/vertexai"; // Add this line

...
</code></pre>
<p>Now we can initialize Gemini through Vertex AI and export it. Add the following lines in the <code>config.ts</code> after initializing the <code>app</code> but before exporting it: </p>
<pre><code class="language-typescript">...

const app = initializeApp(firebaseConfig);

// Add the lines below

// Initialize FirebaseApp
const app = initializeApp(firebaseConfig);

// Initialize the Vertex AI service
const vertexAI = getVertexAI(app);

// Initialize the generative model with a model that supports your use case
const model = getGenerativeModel(vertexAI, { model: "gemini-2.0-flash" });

// Wrap in an async function so you can use await
export async function geminiApi(prompt: string) {
  // To generate text output, call generateContent with the text input
  const result = await model.generateContent(prompt);

  const response = result.response;
  const text = response.text();
  return text;
}

// Add the lines above

export default app;
</code></pre>
